# AGENTS

## Versioning

- `build` must increase by `1` on every commit.
- `major`, `minor`, and `patch` are changed only when explicitly instructed.
- If `major` increases by `1`, reset both `minor` and `patch` to `0`.
- If `minor` increases by `1`, reset `patch` to `0`.
- Git tags for releases should use the format `v<version>`, for example `v0.2.0.20`.
